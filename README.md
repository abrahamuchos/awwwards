# Awwwards Winning Website
Unofficial Zentry landing page. It only serves the purpose of practicing and demonstrating React + Tailwind + Gsap.
*None of its links lead to another page, nor is it possible to make a purchase or other action.*

<img src="/docs/home-screen.png" width="750px" alt="home-screenshot">

Made for demonstration purposes only.

## ✅ Features

- Show Hero section
- Show features
- Show Story

_(All static)._

## ⚙️ Tech Stack

- React 19
- Tailwindcss 3.4
- Vite 6.2
- gsap 3.12
- @gsap/react 2.1
- tailwind-merge 2.6
- react-use 17.6


## 💾 Installation

Install and run

1. Clone and move to folder
```bash
$ git clone git@github.com:abrahamuchos/awwwards.git
$ cd awwwards
```

2. Install dependencies
```bash
$  npm install
```

3. Run dev `npm run dev` or run to production `npm run build`
